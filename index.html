Conhecimentos

HTML

HTML significa Hypertext Markup Language. Ele permite que os usuários criem e estruturem seções, parágrafos, cabeçalhos e links para páginas da internet ou aplicações.

CSS

Utilizando o CSS é possível alterar a cor do texto e do fundo, fonte e espaçamento entre parágrafos, criar tabelas, usar variações de layouts e ajustar imagens para suas respectivas telas.

JavaScript

JavaScript é uma linguagem de programação que permite implementar funcionalidades mais complexas em páginas web. Sempre que uma página web faz mais do que apenas mostrar informações estáticas para você, pode apostar que o Javascript provavelmente está envolvido.

Bootstrap

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web, melhorando a experiência do usuário em um site amigável e responsivo.

Git e GitHub

GitHub é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Ele permite que programadores, utilitários ou qualquer usuário cadastrado na plataforma contribuam em projetos.

PHP

PHP é uma linguagem de programação voltada para o desenvolvimento de aplicações para a web e para criar sites, favorecendo a conexão entre os servidores e a interface do usuário.
